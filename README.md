# Star Trek: Final Frontier — Fan Arcade

Browser-based After Burner-style arcade shooter starring the USS Enterprise-D.

> **Fan project — Non-commercial — Not affiliated with Paramount Pictures / CBS Studios**

## Play

🎮 **[https://facszero.github.io/StarTrekFanProject](https://facszero.github.io/StarTrekFanProject)**

## Controls

| Input | Action |
|---|---|
| Mouse / ← → Arrow keys | Helm (lateral movement) |
| Space / Tap | Launch photon torpedo |
| Auto | Phasers fire continuously |
| ESC | Tactical pause |

## Phase 1 — Space Combat

- USS Enterprise-D (rear ¾ view, banking animations)
- After Burner–style perspective: enemies scale up from the vanishing point
- Auto-targeting phasers + limited homing photon torpedoes
- Enemy types: Klingon Bird of Prey, Borg Cube
- 4-wave progression with escalating formations
- Full LCARS-inspired tactical HUD (shields, radar, score, weapon bar)
- Starfield tunnel effect with warp-speed streaks

## Roadmap

- **Phase 2** — Borg assault level, full LCARS HUD, shield regeneration
- **Phase 3** — Asteroid field, warp chase, tunnel levels
- **Phase 4** — Boss fights (Borg Queen cube, Romulan Warbird)
- **Phase 5** — Sound FX (Web Audio API)

## Tech Stack

Vanilla HTML5 Canvas 2D — zero dependencies, runs anywhere.
